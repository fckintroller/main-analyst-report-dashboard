# 📈 KOREAN ANALYST REPORT

> **"주식 시장의 브레인들이 분석하는 투자 전략을 한눈에 확인하는 올인원 데이터베이스"**

이 프로젝트는 국내 주요 증권사의 최정상급 애널리스트들의 리포트, 투자의견 변동(Upgrade/Downgrade), 그리고 주요 경제 지표를 실시간으로 수집하고 시각화하여 투자 의사결정을 돕는 자동화 대시보드 시스템입니다.

---

## 🚀 주요 기능

### 1. 실시간 데이터 수집 (Crawling & Fetching)
- **네이버 금융 리서치 수집**: 최신 기업/산업 분석 리포트를 자동으로 크롤링하여 데이터베이스화합니다.
- **글로벌 경제 캘린더**: ForexFactory 등의 소스를 통해 시장에 큰 영향을 미치는 'High Impact' 경제 지표 일정을 가져옵니다.
- **주가 데이터 연동**: 네이버 금융 API를 통해 KOSPI 및 개별 종목의 52주 주가 데이터를 패치합니다.

### 2. 스마트 분석 및 리포팅
- **투자의견 추적**: 애널리스트들의 의견 변동(매수, 홀딩, 매도) 및 목표주가 변화를 실시간으로 감지하여 알림을 생성합니다.
- **마크다운 보고서 자동 생성**: 수집된 데이터를 바탕으로 `analyst_awards_report.md` 형태의 정기 보고서를 자동 작성합니다.
- **Google Sheets 연동**: 엑셀 및 구글 시트에서 즉시 활용 가능한 CSV 파일을 생성합니다.

### 3. 인터랙티브 대시보드
- **결합 섹터별 뷰**: 산업간 연관성을 고려한 섹터별 애널리스트 프로필 및 핵심 커버 종목을 확인합니다.
- **수익률 비교 차트**: 코스피 지수 대비 개별 종목의 누적 수익률을 비교 분석할 수 있는 동적 차트를 제공합니다.
- **이벤트 타임라인**: 차트 상에 주요 경제 이벤트를 표시하여 매크로 환경 변화에 따른 주가 변동을 직관적으로 파악합니다.

## 📂 프로젝트 구조

```text
├── crawler.py             # 네이버 금융 리서치 리포트 수집기
├── calendar_fetcher.py    # 글로벌 주요 경제 지표 수집기
├── updater.py             # 데이터 통합 및 대시보드 갱신
├── utils.py               # 공통 경로 및 데이터 처리 유틸리티 (신규)
├── app.js                 # 대시보드 인터랙티브 로직 (신규 분리)
├── index.html             # 메인 웹 대시보드 (UI/UX)
├── fixed_events.json      # 주요 고정 경제 이벤트 데이터 (신규 분리)
├── analyst_database.json  # 메인 데이터베이스
├── economic_calendar.json # 수집된 경제 이벤트 데이터
├── stocks.json            # 종목 마스터 데이터
├── analyst_data.js        # UI 브릿지 파일
├── analyst_table_sheet.csv # CSV 출력물
└── analyst_awards_report.md # 요약 보고서
```

## 🛠 설치 및 실행 방법

### 1. 환경 준비
Python 3.x 버전이 설치되어 있어야 합니다. 별도의 복잡한 라이브러리 설치 없이 파이썬 표준 라이브러리 위주로 구동되도록 설계되었습니다.

### 2. 수동 실행
데이터를 최신 상태로 갱신하려면 아래 순서대로 스크립트를 실행하십시오:
1. `python crawler.py` (리포트 수집)
2. `python calendar_fetcher.py` (경제 지표 수집)
3. `python updater.py` (데이터 통합 및 대시보드 갱신)

### 3. 대시보드 확인
`index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 열면 인터랙티브 대시보드를 바로 확인할 수 있습니다.

### 4. 자동 업데이트 설정 (Windows)
`register_scheduler.bat` 파일을 실행하면 매일 밤 21:00에 자동으로 모든 데이터 수집 및 대시보드 갱신 작업이 수행되도록 윈도우 작업 스케줄러에 등록됩니다.
