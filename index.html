Windows 스케줄러 자동화 가이드

매 2주마다 완전 자동 갱신 환경 구축

1

런처 배치 파일 확인

로컬 폴더 내의 run_scheduler.bat 실행 파일 경로를 확인합니다.

2

Windows 작업 스케줄러 실행

시작 메뉴에서 작업 스케줄러(Task Scheduler)를 실행합니다.

3

새 기본 작업 생성 및 주기 설정

우측의 '기본 작업 만들기'를 눌러 이름을 지정한 후, 주기를 '매월' 또는 '매주'로 설정합니다. (격주 주말 실행 추천)

4

배치 스크립트 연결

동작에서 '프로그램 시작'을 누르고 run_scheduler.bat 경로를 연결합니다.
